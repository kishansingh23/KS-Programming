📚 {Library Management System} (C) Language

A simple Library Management System built using the C programming language.
This project allows users to manage book records efficiently using file handling.

🚀 Features
➕ Add new books
📖 View all books
🔍 Search books by ID or title
❌ Delete books
💾 Store data using file (library.dat)

🛠️ Tech Stack
Language: C
Concepts Used:
 File Handling
 Structures
 Functions
Tools: Git & GitHub

📂 Project Structure
Library-Management-System/
│── main.c
│── library.dat
│── README.md

⚙️ How to Run
1. Clone the Repository
git clone https://github.com/kishansingh23/KS-Programming.git
2. Navigate to Project Folder
cd library-management-system
3. Compile the Code
gcc main.c -o library
4. Run the Program
./library

🎯 How It Works
Run the program
Choose options from the menu
Perform operations like:
Add book
View book list
Search book
Delete book

All records are stored in a file (library.dat).
